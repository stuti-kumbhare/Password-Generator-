# Password Generator Using Python

## Project Overview

The Password Generator is a Python-based application that creates strong and random passwords for users. It allows users to specify the desired password length and generates a secure password using letters, numbers, and special characters.

## Features

* Generates random passwords
* User-defined password length
* Uses uppercase and lowercase letters
* Includes numbers and special characters
* Easy to use

## Technologies Used

* Python
* Random Module
* String Module

## How to Run

1. Open the project folder.
2. Run the following command:

python password_generator.py

3. Enter the desired password length.
4. The generated password will be displayed on the screen.

## Sample Output

Enter the desired password length: 12

Generated Password:
A@7k#P9!xL2$

## Conclusion

This project demonstrates the use of Python libraries such as random and string to generate secure and random passwords efficiently. It helps users create strong passwords for better security.
